bio-scripts
===========

collection of scripts and software tools for bioinformatic analysis

* codonusage
* rbhplus
* fasta
* fastq
* gff
* blastparse

Dependencies
------------
Scripts depends on biopython <http://biopython.org/wiki/Download>, namely the modules:

* Bio.Data
* Bio.SeqIO
* Bio.SeqUtils
* Bio.SeqRecord

Scripts depends on numpy <http://www.numpy.org/>
